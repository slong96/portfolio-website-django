Movie Trivia Quiz

This project was created using HTML, CSS, and JavaScript. It uses Open Trivia Database to generate an API url. The website will fetch the data (questions and answers), then display them. There's also a score counter to keep track of how many right answers the user has. The 'next' button will be enabled when an answer is selected. After answering ten questions, a final score and a 'play again' button will be displayed at the end.
